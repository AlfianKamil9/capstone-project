## Instalation 
If you want to try this project,
1. Clone this project
   
   ```Clone
   git clone https://github.com/AlfianKamil9/capstone-project.git
   ````
2. Install dependencies

   ```Bash
   npm install
   ```
3. Add folder config/config.json

   ```Bash
   {
  "development": {
    "username": "root",
    "password": null,
    "database": "Your-Db-Name",
    "host": "localhost",
    "dialect": "mysql"
    }
  }
   ```
